
## Overview

**Crosshair X** is a lightweight and fully customizable crosshair overlay tool designed for competitive gamers.  
It provides smooth performance, modern UI customization, and works in fullscreen or borderless games without affecting gameplay.

## ⚙️ Core Features

- 🎯 Fully customizable crosshair (size, color, opacity, outline)
- 🖥️ Overlay support for fullscreen and borderless games
- ✨ Built-in animated and static crosshair presets
- ⌨️ Hotkey system (toggle, hide, switch presets)
- ⚡ Lightweight performance (low CPU & RAM usage)
- 💾 Preset saving & loading for different games/playstyles
- 🎮 Smooth rendering engine with minimal input delay
- 🧩 Modern UI with live preview editor
